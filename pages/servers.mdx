# Servers

## Staging server

The staging server is used for testing and development purposes. It is not intended for production use and should not be used in a production environment.

The staging server is located at `https://api.sandbox.ntak.guru`.

Also we use this server for official validation.

## Production server

The production server is used for production purposes. It is the recommended server for production use.

The production server is located at `https://api.ntak.guru`.
