# Deactivate Reporting

This endpoint allows you to deactivate reporting on an accommodation by providing the accommodation `id` or `slug`.

## Request

<Protocol method="POST" path="/v1/accommodations/{accommodation}/deactivate" />

<Tabs items={['cURL', 'PHP']}>
  <Tabs.Tab>
    [//]: # (@formatter:off)
    ```bash copy
    curl -X POST https://api.sandbox.ntak.guru/v1/accommodations/rqdsw/deactivate \
      -H "Authorization: Bearer {token}" \
      -H "Accept: application/json"
    ```
    [//]: # (@formatter:on)
  </Tabs.Tab>
  <Tabs.Tab>
    [//]: # (@formatter:off)
    ```php copy
    use TmrwLife\NtakGuru\Services\Accommodation;

    $gateway = Accommodation::setup([
      'accessToken' => {token},
      'isProduction' => false, // default: false
    ]);

    $response = $gateway->deactivate('rqdsw');
    ```
    [//]: # (@formatter:on)
  </Tabs.Tab>
</Tabs>

### URL Parameters

| Name            | Description                      |
|-----------------|----------------------------------|
| `accommodation` | The accommodation `id` or `slug` |
